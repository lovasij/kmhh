Táblák létrehozása


    CREATE TABLE 112e_ob.paciens (
        pid CHAR(11) PRIMARY KEY,
        nev VARCHAR(50),
        cim VARCHAR(100),
        telefon VARCHAR(20)
    );
    
    CREATE TABLE 112e_OB.vizsgalat (
        vid INT PRIMARY KEY,
        pid CHAR(11),
        datum DATE,
        FOREIGN KEY (pid) REFERENCES PACIENS(pid)
    );
    
    CREATE TABLE 112e_ob.betegseg (
        bnev VARCHAR(50) PRIMARY KEY
    );
    
    CREATE TABLE 112e_ob.gyogyszer (
        gynev VARCHAR(50) PRIMARY KEY
    );
    
    CREATE TABLE 112e_ob.mit_kap (
        bnev VARCHAR(50),
        gynev VARCHAR(50),
        PRIMARY KEY (bnev, gynev),
        FOREIGN KEY (bnev) REFERENCES betegseg(bnev),
        FOREIGN KEY (gynev) REFERENCES gyogyszer(gynev)
    );
    
    CREATE TABLE 112e_ob.mire_erzekeny (
        pid CHAR(11),
        gynev VARCHAR(50),
        PRIMARY KEY (pid, gynev),
        FOREIGN KEY (pid) REFERENCES paciens(pid),
        FOREIGN KEY (gynev) REFERENCES gyogyszer(gynev)
    );
    
    CREATE TABLE 112e_ob.diagnozis (
        vid INT,
        datum DATE,
        bnev VARCHAR(50),
        PRIMARY KEY (vid, bnev),
        FOREIGN KEY (vid) REFERENCES vizsgalat(vid),
        FOREIGN KEY (bnev) REFERENCES betegseg(bnev)
    );

Lekérdezések

1. Listázd azokat a pácienseket, akiknek a neve „K” betűvel kezdődik!


    SELECT nev AS "Név" 
    FROM 112e_ob.paciens
    WHERE nev LIKE 'K%';

2.Mely páciensek NEM érzékenyek semmilyen gyógyszerre?


    SELECT nev "Név" FROM 112e_ob.paciens
    WHERE pid NOT IN (SELECT pid FROM 112e_ob.mire_erzekeny);

3. Hány vizsgálat történt 1996-02-11-én?


    SELECT COUNT(*) "Vizsgálatok száma" FROM 112e_ob.vizsgalat
    WHERE datum = '1996-02-11';

4. Listázd a páciensek nevét és a diagnosztizált betegségeiket!


    SELECT p.nev " Páciens neve", d.bnev "Betegség neve" FROM 112e_ob.paciens p
    JOIN 112e_ob.vizsgalat v ON p.pid = v.pid
    JOIN 112e_ob.diagnozis d ON v.vid = d.vid;

5. Hány különböző betegséget diagnosztizáltak egyes pácienseknél?


    SELECT p.nev "Páciens neve", COUNT(DISTINCT d.bnev) "Betegségek száma"
    FROM 112e_ob.paciens p
    JOIN 112e_ob.vizsgalat v ON p.pid = v.pid
    JOIN 112e_ob.diagnozis d ON v.vid = d.vid
    GROUP BY p.nev;

6. Mely vizsgálatokhoz tartozik több mint egy diagnózis?


    SELECT DISTINCT p.nev, me.gynev FROM 112e_ob.paciens p
    JOIN 112e_ob.vizsgalat v ON p.pid = v.pid
    JOIN 112e_ob.diagnozis d ON v.vid = d.vid
    JOIN 112e_ob.mit_kap m ON d.bnev = m.bnev
    JOIN 112e_ob.mire_erzekeny me 
        ON p.pid = me.pid AND m.gynev = me.gynev;

7. Hány páciens érzékeny egyes gyógyszerekre?


    SELECT v.vid "Vizsgálat-id", COUNT(d.bnev) "Betegségek száma"
    FROM 112e_ob.vizsgalat v
    JOIN 112e_ob.diagnozis d ON v.vid = d.vid
    GROUP BY v.vid
    HAVING COUNT(d.bnev) > 1;

8. Listázd azokat a betegségeket, amelyeket legalább 2 különböző vizsgálat során diagnosztizáltak!


    SELECT g.gynev "Gyógyszer neve", COUNT(me.pid) "Érzékenyek száma"
    FROM 112e_ob.gyogyszer g
    JOIN 112e_ob.mire_erzekeny me ON g.gynev = me.gynev
    GROUP BY g.gynev;

9. Mely pácienseknek volt pontosan 1 vizsgálatuk?


    SELECT d.bnev "Betegség neve", COUNT(DISTINCT d.vid) "Diagnózisok száma"
    FROM 112e_ob.diagnozis d
    JOIN 112e_ob.betegseg b ON d.bnev = b.bnev
    GROUP BY d.bnev
    HAVING COUNT(DISTINCT d.vid) >= 2;

10. Listázd ki a pácienseket, a betegségeiket és a gyógyszereiket!


    SELECT p.nev "Beteg neve"
    FROM 112e_ob.vizsgalat v
    JOIN 112e_ob.paciens p ON v.pid = p.pid
    GROUP BY v.pid
    HAVING COUNT(*) = 1;
